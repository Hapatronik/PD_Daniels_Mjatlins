# PD_Daniels_Mjatlins
Cementa kalkulators

Šis kalkulators aprēķina nepieciešamo smilšu un ūdens daudzumu, kā arī kopējās izmaksas, ja ir zināma cena un gala betona masu.

Lai palaist programmu, ir jāuzstāda programmas visual studio code un python, Tad jāatver fails visual studio code, jāuzspiež taustiņš f5, un jāievada nepieciešamos datus, tad jāuzspiež enter. Programma izdos aprēķinātos rezultātus.
